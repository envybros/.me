# 현재 우리는 알고리즘 튜토리얼을 작성 및 번역을 하고 있다.

## 당신에게 요청하는 내용

1. **아래 내용 번역해주길 바란다**. 
2. **한국인**이 이해하기 어려운 표현이나 단어는 피해주길 바란다.
2. "`" <- 이런 기호가 보이면 "'" <- 이러한 형식으로 고쳐야 한다.
3. 우리는 **절대적으로** "공식적인 언어 형식"으로 진행할 필요가 있다("이다.", "한다." <- 이러한 형태, "입니다.", "합니다." <- 이런 형태는 사용하지 말자.).
4. Latex 표기법이 깨졌다면, 정상적으로 보일 수 있도록 수정이 필요하다($n$ <- 이런 식으로 $ 기호는 각 하나씩 사용하도록 하자).
5. 문맥에 맞게 문단을 분리해야 할 필요가 있을 경우, 문단을 분리해도 좋다.
6. 변수명, 함수명, true, false와 같은 예약어는 번역하면 안된다.

- subarray: 부분 배열

- window: 윈도우

- prefix sum: 구간 합

- data structures: 자료 구조
